Research Focus

Learning spatial correspondence from imperfect views.

My current work explores how models can localize a moving camera using satellite imagery, partial observations, cached visual tokens, and trajectory-level constraints. The long-term goal is to make geo-localization systems that understand place as structure, not just as nearest-neighbor image retrieval.

Research Directions

01

Cross-view localization

Matching street-level trajectories to overhead imagery while handling viewpoint gaps, satellite coverage quality, and hard-site ambiguity.

02

Spatial memory tokens

Exploring memory-like representations that accumulate evidence across views and expose where a model is confident, repetitive, or collapsed.

03

Foundation visual features

Using DINO-style visual tokens for ground and satellite encoders, with careful separation between cached-feature pipelines and in-loop training.

04

Trajectory-aware reasoning

Combining retrieval, offset refinement, temporal consistency, and path shape so localization uses motion as evidence instead of metadata alone.
